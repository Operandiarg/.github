# Operandi

**We build software that people actually want to use.**

Operandi is a startup studio based in Buenos Aires, Argentina. We design and build B2B SaaS products and custom automation systems for companies that are tired of duct-taping their processes together.

---

### What we do

We work across the full product lifecycle — from figuring out what to build, to designing how it should feel, to shipping production code. Our focus areas:

- **Custom software development** — Full-stack products built with intention, not just functionality.
- **Process automation** — We turn repetitive, manual workflows into systems that run themselves.
- **AI-native solutions** — Agents, pipelines, and tools that use AI where it actually makes sense, not where it sounds cool.

### How we think

We believe that a product nobody enjoys using is just software. And software is everywhere.

What's rare is something that feels like it was made for you. That's what we aim for — every time.

Before writing a single line of code, we define how the person using the product should _feel_. Then we design. Then we build.
